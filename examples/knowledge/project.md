# Project Context

## Domain

Educational Technology / AI in Education

## Description

Writing an academic paper that explores how context engineering techniques,
implemented through ADL (Assistant Description Language), can be used to
create personalized educational tutors. The paper will:

1. Define context engineering in the LLM era
2. Present ADL as a context specification language
3. Show how ADL enables personalized learning experiences
4. Provide examples of educational tutors built with ADL
5. Discuss implications for adaptive learning systems
